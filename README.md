# My To-Do List

A simple, beautiful, and responsive to-do list web application built with HTML, CSS, and JavaScript.

## Features

- **Add Tasks**: Easily add new tasks to your list by typing in the input field and clicking "Add Task".
- **Delete Tasks**: Remove tasks from the list by clicking the "Delete" button next to each item.
- **Clean UI**: Modern, user-friendly interface with smooth transitions and hover effects.
- **Responsive Design**: Works well on both desktop and mobile devices.
- **No Empty Tasks**: Prevents adding empty or whitespace-only tasks.

## Technologies Used

- **HTML5**: Structure of the web page
- **CSS3**: Styling and layout (Flexbox, transitions)
- **Vanilla JavaScript**: Functionality for adding and deleting tasks

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Type a task in the input field.
4. Click the "Add Task" button to add it to your list.
5. Click the "Delete" button next to any task to remove it.
